# 日本語学习笔记

令和四年。歲在壬寅。

欲學日語。書此筆記。

一字一符。冀有所得。

若無獲益。亦毋悲己。

心平氣清。恆唸辭句。

秦穆飲盜馬。楚客報絕纓。
